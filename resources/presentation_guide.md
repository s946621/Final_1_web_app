# Video Script Guide (2–3 Minutes)

## 1. Problem Statement (≈30–45 seconds)
*Goal: Clearly explain the problem and who it affects.*

- “The problem we are trying to solve is…”
- “This problem matters because…”
- “The people most affected by this problem are…”
- “Right now, this problem creates challenges such as…”
- “A real-world example of this problem is…”

---

## 2. Current Solutions Fall Short, because… (≈45–60 seconds)
*Goal: Show awareness of existing solutions and their limitations.*

- “Some current solutions to this problem include…”
- “These solutions attempt to solve the problem by…”
- “However, these solutions fall short because…”
- “One limitation is…”
- “Another issue is…”
- “Because of these limitations, users still struggle with…”

---

## 3. Solution Description (≈60–75 seconds)
*Goal: Demonstrate the software and how it improves the situation.*

- “Our solution is called…”
- “Our solution works by…”
- *(Begin screen-share walkthrough)*  
  - “Here is the main screen where users can…”
  - “When a user clicks on…, it allows them to…”
  - “This feature helps solve the problem by…”
  - “Another key feature is…”
- “What makes our solution different is…”
- “Compared to existing solutions, our software improves by…”
- “This matters because…”

---

## 4. Code Spotlight (Each Student Speaks) (≈45–60 seconds total)
*Goal: Highlight technical thinking and individual contributions.*

Each student involved in the project must briefly share one piece of code they are proud of.

- “A piece of code I’m proud of is…”
- “This code is responsible for…”
- *(Show code on screen)*  
- “What this code does is…”
- “The most important part of this code is…”
- “One challenge I faced while writing this was…”
- “I solved that challenge by…”
- “This improves our program because…”

### Optional (Advanced):
- “This code uses [loop/condition/function/API/etc.] to…”
- “An alternative approach could have been…, but I chose this because…”
- “This makes the program more efficient/readable/scalable because…”

---

## 5. Closing (≈15–20 seconds)
*Goal: Reinforce impact and future thinking.*

- “In summary, our solution addresses the problem by…”
- “In the future, we could improve this by…”
- “Ultimately, this solution could help users by…”

---

## Suggested Timing Breakdown

- Problem: ~30–45 sec  
- Current Solutions: ~45–60 sec  
- Solution Demo: ~60–75 sec  
- Code Spotlight: ~45–60 sec total  
- Closing: ~15–20 sec
